# BLACK AND YELLOW - Wireframes

> Private Perpetuals Exchange - Page Wireframes

---

## Page Structure

```
/                    → Markets Dashboard (landing)
/trade/:pair         → Trading Interface (e.g., /trade/btc-usdc)
/deposit             → Cross-Chain Deposit (LI.FI)
/positions           → Position Management
/history             → Trade History
```
